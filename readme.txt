== Description ==

Adds custom fields to the BCD Roster custom post type along with a new shortcode for displaying a new roster.

**Shortcode(s):**

* **[bcdrc]** : Displays a list of all roster members using the new roster companion layout

For detailed usage instructions please visit [duhjones.com](http://duhjones.com/downloads/bcd-roster-companion/)

== Installation ==

1. Upload plugin to a folder named 'bcd-roster-companion' within the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update existing roster members or create new ones using the newly added fields

== Frequently Asked Questions ==

= Are there any shorcodes available to display the new fields? =

Yes.  The [bcdrc] shortcode can be used to display the new fields.  Visit [duhjones.com](http://duhjones.com/downloads/bcd-roster-companion/) for details on how to use the shortcode.
